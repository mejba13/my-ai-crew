### Prerequisites

- [Claude Code](https://docs.claude.com/s/claude-code) installed
- Claude Max or API access (Opus 4.5 recommended)

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/mejba13/my-ai-crew.git
   cd my-ai-crew
   ```

2. **Launch Claude Code**
   ```bash
   claude
   ```

3. **Verify agents are loaded**

   ```bash
   /agents
   ```

4. **Start creating content**
   ```bash
   @aria Create a blog post for mejba.me about [your topic]
   ```
